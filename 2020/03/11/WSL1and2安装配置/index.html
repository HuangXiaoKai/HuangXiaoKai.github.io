WSL and WSL2搭建
发表于 2020-03-11| 更新于 2020-03-16|CS
阅读量:

已经是2020年了，用Windows10的小伙伴应该要听说Windows Subsystem Linux(WSL)了，这个软件让我们能在不装虚拟机的情况下，在Windows系统中以“优雅”的方式使用Linux。这对于我这种使用Linux仅限于小众科学计算软件，对GUI需求不大，不需要Linux桌面，而同时又完全舍不得Microsoft全家桶的人来说可以说是非常完美的计算机配置方案了。

本人安装WSL经历先用WSL，再因为某些Linux软件GUI的要求而升级WSL2的过程。其中也尝试过对Terminal进行美化，这里就将我经历的操作总结下来。

查阅Windows系统的版本

只是以防万一，确认大家都知道如何查阅自己Windows 10系统的版本信息。请打开系统设置，然后：

（会一点英语很容易将这些单词翻译成中文）

图中可以方便看到我的电脑Version 2004（2020年4月的意思），OSbuild 19041.113。在今天（2020年3月）你的Version还是1909也是很正常的，只要你没有选择Insider program（后面说WSL2时会提到）Windows更新不一定会第一时间推送到你的电脑上；但是如果你的Version还是1903，建议你去Settings -> Update & Security -> Windows Update中刷新更新信息并安装更新。
